# Financial Calculator

This project is a basic financial calculator website created using HTML, CSS and JavaScript.
It was developed as a practice project to understand how financial calculations and simple user interfaces can be implemented in web development.

The project includes different calculator pages such as EMI, SIP, GST, tax and investment related calculations.
Some parts of the project are still under development and may not work perfectly.

# Purpose of the Project

The main aim of building this project was:

To practice JavaScript logic building

To understand financial calculation formulas

To learn multi-page website structure

To improve frontend development skills

This is mainly a learning project and not a complete production-ready application.

# Current Features

Simple landing page for navigation

Separate pages for different financial calculators

Basic UI design using CSS

User input based calculation logic

Beginner-friendly project structure

# Some calculators may not give accurate results or may be incomplete.
